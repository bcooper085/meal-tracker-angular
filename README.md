# _Meal Tracker_
#### Project was completed under 2 different user logins resulting in a loss of commits.

### Enter a meal, meal details and calories to track your calorie intake.

## By Brandon Cooper

# Description
A mock site to enter meals and track calories and fresh "healthy meals"

# Specifications

- User can input a meal
- User can add details and calories to meal
- Meal will be displayed by calorie filter
- Meal list will be filtered by all meals and fresh meals
